# 📂 Research Project Archives

Welcome to our research file storage! Since we are using GitHub to keep track of our **20+ versions** of the paper, please read this quick guide on how to get the files you need.

---

### 📥 How to Download the LATEST Version (Easiest)
If you just want the most recent, "ready-to-edit" version:
1. Look at the right-hand sidebar of this page under **"Releases"**.
2. Click on the latest version (e.g., **v15.9-patched**).
3. Under the "Assets" section, click the **.docx** file to download it instantly.

---

### 📜 How to Download OLDER Versions
If you need an older draft (like v13.5 or an unpatched version):
1. Stay on the main page of this repository.
2. Scroll through the list of files until you find the one you want.
3. **Click the filename** (e.g., `RESEARCH-V13.5.docx`).
4. Look for the **Download icon** (it looks like a small downward arrow ⬇️) on the top right.
5. Click it, and it will save to your computer.

---

### ⚠️ IMPORTANT: DO NOT EDIT ONLINE
* **Do not** try to edit these files directly on GitHub (it won't work for Word docs).
* **Download** the file → **Edit** on your computer → **Send it back** to Adrian so he can upload the new version here.

---

### 🛠 Troubleshooting
* **"I see weird code!"** – You probably clicked a file that GitHub is trying to read as text. Just look for the Download button/arrow or the "View Raw" button to get the actual Word file.
* **"Which one is the newest?"** – Always check the **Releases** section first; that is where the "official" latest draft will live.# Research-Archives
